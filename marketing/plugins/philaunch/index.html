PhiLaunch

Remote automation that just works. SSH orchestration, real-time monitoring, portable configuration.

For everyone. From basic users to advanced hackers. Adaptive complexity. Zero bullshit.

Download Quick Start View on GitHub

The Problem

• Hardcoded IPs, usernames, paths in scripts

• No visibility into remote system status

• Manual SSH from phone is painful

• No testing infrastructure

• Credentials scattered everywhere

The Solution

✓ Central config with setup wizard

✓ Real-time web dashboard

✓ Android SSH via WireGuard VPN

✓ Complete CI/CD pipeline

✓ Secure credential management

Features

Practical tools. No promises. Outcomes delivered.

Portable Configuration

Central philaunch.conf eliminates hardcoded values. Setup wizard auto-detects environment. All scripts load config automatically. Change once, update everywhere.

$ ./setup.sh
$ source config/load-config.sh

Real-Time Monitoring Dashboard

Web-based UI shows CPU, memory, disk, network, tmux sessions, WoW server latency, recent logs. Auto-refresh every 5 seconds. Dark theme. Mobile-responsive. Pure HTML/CSS/JS.

$ ./start-dashboard.sh
# http://localhost:8080

CI/CD Pipeline

GitHub Actions with 7 automated checks: ShellCheck, security scan (TruffleHog/GitLeaks), syntax validation, bats test suite, dependency check, config validation, markdown lint. Pre-commit hooks prevent bad commits.

$ ./tests/run-tests.sh
$ git commit # hooks run

Android SSH Integration

Complete guide for SSH from Android 11 to Windows 11 via WireGuard VPN. Termux setup, ED25519 keys, router configuration, firewall rules. Phone-to-desktop automation.

# See docs/ANDROID-SSH-SETUP.md
$ ssh user@192.168.1.X

Identity Separation System

Clear role definitions for multi-Claude instances (WEB_CLAUDE, TERMINAL_CLAUDE, VS_CODE_CLAUDE). Decision matrix prevents task overlap. Session-start prompts ensure identity awareness.

# claude-bootstrap/IDENTITY-PROFILES.md
# claude-bootstrap/SESSION-START-PROMPTS.md

Comprehensive Testing

Bats test framework with unit and integration tests. Test utilities with mocking functions. Dependency checker validates required tools. Documentation for test writing and troubleshooting.

$ bats tests/unit/*.bats
$ scripts/check-dependencies.sh

Tech Stack

Core: Bash, Python, SSH, tmux

Dashboard: HTML/CSS/JS, Python HTTP server

Testing: Bats, ShellCheck, TruffleHog, GitLeaks

Network: WireGuard VPN, SSH keys, iptables

CI/CD: GitHub Actions, pre-commit hooks

Deployment

Works on Linux, macOS, WSL, Termux (Android). Minimal dependencies—standard tools only. Setup wizard handles platform detection. Portable across environments.

Quick Start

Clone. Run setup. Done.

Installation

# Clone repository
git clone https://github.com/Stryk91/PhiLaunch.git
cd PhiLaunch

# Run setup wizard
./setup.sh

# Start dashboard
./start-dashboard.sh

# Access dashboard
# http://localhost:8080

Configuration

# Manual config (optional)
cp config/philaunch.conf.example \\
   config/philaunch.conf

# Edit configuration
nano config/philaunch.conf

# Validate config
source config/load-config.sh

# Run tests
./tests/run-tests.sh
