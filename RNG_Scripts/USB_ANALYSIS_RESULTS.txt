========================================
THE PROBLEM
========================================

Your 50GB Phone Link transfer is going through the cloud because:

1. Your phone IS connected via USB (correct)
2. BUT it's on the Intel USB 3.0 port (5 Gbps)
3. NOT on the ASMedia USB 3.1 port (10 Gbps)
4. Phone Link doesn't recognize USB as "fast enough" for local transfer
5. So it defaults to cloud relay (Microsoft servers)

Even if you switched ports, Phone Link might STILL use cloud
because Microsoft's software is stupid and doesn't properly
detect USB connections for local transfer.


========================================
HOW TO FIX THIS
========================================

STEP 1: Find Your USB 3.1 Ports
--------------------------------

Your ASMedia USB 3.1 ports are usually:
- On the motherboard I/O panel (back of PC)
- Color: Usually RED or CYAN/TURQUOISE
- OR labeled "USB 3.1" or "10G" or "SS+"

Your Intel USB 3.0 ports are usually:
- Also on motherboard I/O panel
- Color: Usually BLUE
- OR labeled "USB 3.0" or "SS"

TO IDENTIFY WHICH IS WHICH:
1. Unplug your phone
2. Run USB_ANALYZER_MENU.bat on Desktop
3. Choose option [1] Quick Check
4. This opens Device Manager
5. Expand "Universal Serial Bus controllers"
6. Plug phone into DIFFERENT physical USB ports one at a time
7. Watch which controller shows activity
   - If "Intel" = USB 3.0 (5 Gbps)
   - If "ASMedia" = USB 3.1 (10 Gbps) ✓ USE THIS ONE

STEP 2: Switch to Correct Port
-------------------------------

1. Unplug phone from current USB port
2. Find a port connected to ASMedia controller
3. Plug phone into ASMedia USB 3.1 port
4. Check Device Manager to verify it's on ASMedia now

STEP 3: Still Won't Help Phone Link!
-------------------------------------

Even with the correct port, Phone Link is BROKEN for large files.
It will STILL route through the cloud.

INSTEAD, use one of these methods:

METHOD 1: Direct USB File Transfer (FASTEST)
  1. Phone plugged into USB 3.1 port
  2. On phone: Swipe down, tap "USB for file transfer"
  3. Select "File Transfer" mode
  4. On PC: Open File Explorer
  5. Phone appears as device
  6. Drag and drop 50GB file
  7. DONE in 1-2 minutes at ~1,000 MB/s

METHOD 2: Nearby Share / Quick Share
  1. Turn on "Nearby Share" on PC (Windows settings)
  2. Turn on "Quick Share" on Samsung phone
  3. Right-click file on PC → Share → Choose phone
  4. Uses WiFi Direct (fast, but slower than USB)
  5. ~100-500 MB/s depending on WiFi

METHOD 3: Fix Phone Link (Advanced)
  1. Run OPEN_PHONELINK_FIREWALL.bat (on Desktop)
  2. Opens ports 47000-48000 for direct P2P
  3. Restart Phone Link app
  4. MIGHT work now, but no guarantee
