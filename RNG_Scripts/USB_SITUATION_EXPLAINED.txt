========================================
WHY ASMEDIA UNDERPERFORMS:
========================================

ASMedia USB controllers are notorious for:

1. OUTDATED DRIVERS
   - Rarely updated via Windows Update
   - Must get from motherboard manufacturer
   - Old drivers = poor performance

2. FIRMWARE BUGS
   - ASMedia 1042/1142/1143/1242 chipsets have bugs
   - Causes speed throttling
   - Only fixable via BIOS updates

3. UASP MODE ISSUES
   - UASP (USB Attached SCSI) sometimes slower than BOT
   - Device-specific problem
   - Can disable UASP to fix

4. POWER DELIVERY PROBLEMS
   - Can't reliably deliver 900mA to devices
   - Devices may fall back to USB 2.0 mode
   - Causes intermittent disconnects

5. CHIPSET CONFLICTS
   - ASMedia is add-on controller to Intel chipset
   - Sometimes conflicts with Intel xHCI
   - BIOS settings can help

IF YOU WANT TO FIX ASMEDIA (Optional):
========================================

Try these in order:

[FIX 1] Update ASMedia Drivers
  1. Find motherboard model: Run FIX_ASMEDIA_USB.bat → Option 2
  2. Visit manufacturer website (ASUS/MSI/Gigabyte/ASRock)
  3. Download latest "USB 3.1 Driver" or "ASMedia USB Driver"
  4. Install and reboot
  5. Test speed again with Samsung Magician

[FIX 2] Update BIOS/UEFI
  1. Check current BIOS version: Run FIX_ASMEDIA_USB.bat → Option 5
  2. Visit motherboard manufacturer website
  3. Download latest BIOS (check changelog for USB fixes)
  4. Follow manufacturer's update procedure (BE CAREFUL!)
  5. Test speed again

[FIX 3] Disable UASP Mode
  1. Run FIX_ASMEDIA_USB.bat → Option 3
  2. Follow instructions to uninstall UASP driver
  3. Replug device (reinstalls as BOT mode)
  4. Test speed again

[FIX 4] Check BIOS USB Settings
  1. Reboot into BIOS/UEFI (usually DEL or F2 during boot)
  2. Look for "Advanced" → "USB Configuration"
  3. Try these settings:
     - USB 3.1 Support: Enabled
     - XHCI Hand-off: Enabled
     - Legacy USB Support: Disabled
  4. Save and reboot
  5. Test speed again
